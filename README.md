# 💎 DIGITAL FIKY STORE - PPOB SYSTEM (V145) 💎

Selamat datang di repositori resmi **DIGITAL FIKY STORE**. Script Aplikasi Web PPOB modern, super cepat, dan *auto-pilot* dengan integrasi Digiflazz & WhatsApp Bot (Baileys).

## ✨ Fitur Unggulan (V145 Ultimate)
- **Sistem 3 Bot Telegram:** Notifikasi terpisah dan sangat detail untuk Transaksi (Order Masuk/Sukses/Gagal), Top Up Saldo, dan Auto-Backup Server.
- **Keamanan Berlapis:** Login, Daftar, dan Ganti Password/Nomor wajib menggunakan kode OTP via WhatsApp.
- **Kategori Produk Terlengkap:** Pulsa, Data (Daftar Lengkap Provider), Game, Voucher, E-Wallet, PLN, Masa Aktif, Perdana, E-Toll, dan Tagihan PPOB.
- **Smart Custom Margin (Menu 9):** Atur keuntungan otomatis berdasarkan tier harga modal langsung dari Digiflazz.
- **Manajemen Banner Lokal (Menu 8):** Upload gambar banner promo langsung ke VPS tanpa takut link *expired*.
- **Panel VPS 20 Menu:** Manajemen server, saldo member, edit produk lokal, sinkronisasi produk, dan setting API sangat mudah, cukup dengan 1 perintah di terminal.
